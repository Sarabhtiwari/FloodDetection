Sensor Data (POST) ---> Flask Backend ---> ML Model ---> Prediction
                                              |
                                              V
                                       Store in Database


uml code
@startuml
title Flood Detection System Flow

start

:IoT Sensors collect data;
note right
  Water level
  Rainfall
  Temperature
end note

:Send data to Backend (Flask API);

:Backend sends data to ML Model;

:ML Model predicts flood risk;
note right
  Safe / Warning / Flood Risk
end note

:Backend receives prediction;

if (Store prediction?) then (yes)
  :Store in database;
endif

:Send prediction + readings to Frontend;

:Frontend displays data;
note right
  Color-coded alerts:
  Green: Safe
  Yellow: Warning
  Red: Flood Risk
end note

stop
@enduml
